# RxScan Technologies Website

Site statique officiel de RxScan Technologies Inc., compatible avec GitHub Pages.

## Fichiers

- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Publication GitHub Pages

1. Déposer tous les fichiers à la racine du dépôt GitHub.
2. Ouvrir les paramètres du dépôt.
3. Aller dans Pages.
4. Choisir la branche principale et le dossier racine.
5. Enregistrer.
