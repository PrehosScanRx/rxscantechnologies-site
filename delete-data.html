Suppression des données — RxScan Paramedic

Demande publique de suppression des données pour les utilisateurs de RxScan Paramedic.

Les utilisateurs peuvent demander la suppression des données liées à RxScan Paramedic en communiquant avec RxScan Technologies Inc.

Comment faire une demande

  1. Envoyez un courriel à contact@rxscantechnologies.ca.
  2. Indiquez Demande de suppression des données RxScan Paramedic dans l'objet du message.
  3. Précisez l'adresse courriel utilisée dans l'application ou pour l'envoi de rapports, si applicable.
  4. Ajoutez toute information utile permettant d'identifier les rapports ou données concernés, comme un identifiant de scan si vous en disposez.

Données pouvant être supprimées

  • Rapports PDF envoyés ou conservés côté serveur, si applicables.
  • Données associées aux rapports transmis.
  • Adresses courriel utilisées pour l'envoi ou la réception de rapports, si applicables.

Données stockées localement

Les données stockées uniquement localement sur l'appareil peuvent être supprimées directement dans l'application via l'option Suppression des données, lorsque disponible, ou en désinstallant l'application.

Conservation temporaire

Certaines données minimales peuvent être conservées temporairement lorsque cela est nécessaire pour des raisons légales, de sécurité ou de conformité.

Data Deletion — RxScan Paramedic

Public data deletion request page for RxScan Paramedic users.

Users may request deletion of data related to RxScan Paramedic by contacting RxScan Technologies Inc.

How to submit a request

  1. Send an email to contact@rxscantechnologies.ca.
  2. Use RxScan Paramedic data deletion request as the subject line.
  3. Specify the email address used in the application or for report submission, if applicable.
  4. Include any useful information that may help identify the reports or data concerned, such as a scan identifier if available.

Data that may be deleted

  • PDF reports submitted or stored server-side, if applicable.
  • Data associated with submitted reports.
  • Email addresses used to send or receive reports, if applicable.

Locally stored data

Data stored only locally on the device can be deleted directly in the application through the Data Deletion option, when available, or by uninstalling the application.

Temporary retention

Some minimal data may be retained temporarily when necessary for legal, security, or compliance reasons.
